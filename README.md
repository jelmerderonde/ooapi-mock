# OOAPI MOCK server

This is a simple mock server based on WireMock. It supports the following OOAPI v4 requests:

- /
- /organizations
- /organizations/7bc05553-4b68-44e8-b7bc-37be63c6d9e9
- /buildings
- /buildings/fee3295a-6e55-45b0-a091-7b4de7e39de7
- /courses
- /courses/a6aaec24-4fe5-49f2-8b74-509d2f931acc

None of the OOAPI request parameters are supported. This server will just return the same json response regardless of which paramaters are set.
